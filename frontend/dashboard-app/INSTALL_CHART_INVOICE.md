# Library yang Diperlukan untuk ChartInvoice

## Library Dependencies

### 1. Core Libraries (React & Chart)
```bash
npm install react@^18.3.1 react-dom@^18.3.1
npm install chart.js@^4.5.1
npm install react-chartjs-2@^5.3.1
```

### 2. Material-UI Libraries
```bash
npm install @mui/material@latest
npm install @mui/icons-material@latest
npm install @emotion/react@^11.14.0
npm install @emotion/styled@^11.14.1
```

### 3. Date Picker Libraries
```bash
npm install @mui/x-date-pickers@latest
npm install dayjs@^1.11.19
```

### 4. Development Dependencies
```bash
npm install --save-dev vite@^5.0.0
npm install --save-dev @vitejs/plugin-react@^4.3.1
```

## Perintah Install Lengkap (Sekaligus)

### Menggunakan npm:
```bash
npm install react@^18.3.1 react-dom@^18.3.1 chart.js@^4.5.1 react-chartjs-2@^5.3.1 @mui/material @mui/icons-material @emotion/react@^11.14.0 @emotion/styled@^11.14.1 @mui/x-date-pickers dayjs@^1.11.19
npm install --save-dev vite@^5.0.0 @vitejs/plugin-react@^4.3.1
```

### Menggunakan yarn:
```bash
yarn add react@^18.3.1 react-dom@^18.3.1 chart.js@^4.5.1 react-chartjs-2@^5.3.1 @mui/material @mui/icons-material @emotion/react@^11.14.0 @emotion/styled@^11.14.1 @mui/x-date-pickers dayjs@^1.11.19
yarn add -D vite@^5.0.0 @vitejs/plugin-react@^4.3.1
```

### Menggunakan pnpm:
```bash
pnpm add react@^18.3.1 react-dom@^18.3.1 chart.js@^4.5.1 react-chartjs-2@^5.3.1 @mui/material @mui/icons-material @emotion/react@^11.14.0 @emotion/styled@^11.14.1 @mui/x-date-pickers dayjs@^1.11.19
pnpm add -D vite@^5.0.0 @vitejs/plugin-react@^4.3.1
```

## Daftar Library yang Digunakan:

1. **react** & **react-dom** - Framework React untuk komponen UI
2. **chart.js** - Library charting utama untuk membuat grafik
3. **react-chartjs-2** - Wrapper React untuk Chart.js
4. **@mui/material** - Material-UI component library untuk UI components
5. **@mui/icons-material** - Icon library untuk Material-UI
6. **@emotion/react** & **@emotion/styled** - CSS-in-JS library yang diperlukan MUI
7. **@mui/x-date-pickers** - Date picker components dari MUI X
8. **dayjs** - Library untuk manipulasi tanggal/waktu
9. **vite** & **@vitejs/plugin-react** - Build tool dan plugin React untuk development

## Catatan Penting:

- Pastikan Node.js terinstall (versi 16 atau lebih baru)
- Pastikan berada di direktori `frontend/dashboard-app` sebelum menjalankan perintah install
- Library akan otomatis terdaftar di `package.json` setelah install
